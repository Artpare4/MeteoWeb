## Installation du projet:
### Prérequis:
- Installer **composer**
- Installer **Symfony**

### Installaton:

1. Installer les dépendances requises pour le projet:
```
composer install
```

2. Générer la base de donnée
```
composer db
```

3. Lancer le serveur web local
```
composer start
```
